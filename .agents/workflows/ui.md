# ui
UIデザインのモックアップ画像を生成し、その後にコードとして実装します。

## ルール (AI向けインストラクション)
1. **[画像生成の優先]** ユーザーから「〇〇のUIを作って」と依頼されたら、いきなりHTML/CSSを書かず、**必ず最初に `generate_image` ツールを使ってデザインのモックアップ画像を生成**すること。
2. **[デザイン方針]** ユーザーからの特別な指定がない限り、モダンでリッチなデザイン（ダークモード、グラスモーフィズム、サイバーパンクテイスト等）を意識すること。
3. **[ユーザー確認]** 生成した画像（画像へのリンク）をユーザーに提示し、「このデザインテイストで実装を進めてよいか」を確認すること。
4. **[実装]** ユーザーの承認が得られたら、HTML, CSS, JavaScriptを用いて、モックアップに忠実な実装を行うこと。

## 出力フォーマット（フェーズ1: デザイン提案時）
```markdown
# 🎨 UIモックアップのご提案

ユーザーの要望に基づき、UIのデザインモックアップを作成しました。
![UIモックアップ](生成した画像の絶対パス)

## デザインのポイント
- [配色やレイアウトの意図]
**このデザインの方向性で実際のHTML/CSS/JSの実装に進めてもよろしいでしょうか？**
```
