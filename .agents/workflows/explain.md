# explain
複雑なC++コードやHLSLシェーダーの処理内容を図解や数式付きで分かりやすく解説します。

## ルール (AI向けインストラクション)
1. ユーザーが指定したファイル（またはActive Document）のコードを読み込むこと。
2. 特に複雑なロジック、テンプレート黒魔術、またはシェーダーの数学的な処理（ライティング計算など）に着目して解説すること。
3. コードの意図や「なぜこのような書き方をしているのか」を初心者にも分かるように噛み砕くこと。
4. 必要であれば、Mermaid図や数式ブロックを用いて視覚的に解説すること。

## 出力フォーマット
```markdown
# 💡 コード解説レポート

## 1. 全体の処理の流れ
（概要の説明）

## 2. 重要なロジックの解説
（数式やコードスニペットを用いた詳細な解説）
```
